
Zavr – UI Screenshots

This folder contains screenshots from Zavr, a personal budgeting iOS application currently under active development.

The purpose of these images is to demonstrate:
	•	User interface design
	•	Core budgeting workflows
	•	Attention to usability, clarity, and edge cases
	•	Real-world application development in SwiftUI

About Zavr

Zavr is a privacy-focused budgeting app built for users who want a simple, offline-first way to manage their money without bank connections, subscriptions, or AI dependencies.

The app focuses on:
	•	Manual budget planning
	•	Clear paycheck allocation
	•	Short-term, long-term, and holiday savings goals
	•	Straightforward daily spending awareness
